## Connection methods

MeterMonitor supports three ways to link Stripe in **Settings > Stripe Account Connection**:

1. **Install/connect Stripe now** (OAuth)
2. **Claim existing install** (already installed, enter `account_id`)
3. **Use Stripe restricted key** (`rk_test_...` or `rk_live_...`)

## Connect with OAuth

1. Click **Connect Stripe** or **Change Stripe Connection**.
2. Select **I will install/connect Stripe now**.
3. Complete Stripe authorization.
4. Return to MeterMonitor.

## Verify connection status

In **Stripe Account Connection**, confirm:

- **Connected** badge
- **Environment** (`Test` or `Live`)
- **Auth** method (`OAuth` or `Restricted Key`)
- Stripe account ID (`acct_...`)

Click **Refresh Status** if the view is stale.

## Switch account or auth method

1. Click **Change Stripe Connection**.
2. Select the method you want to use.
3. Complete the selected flow.

MeterMonitor keeps a single active mode at a time (`test` or `live`) for ingestion.
