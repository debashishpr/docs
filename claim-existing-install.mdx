---
title: "Claim existing install"
description: "Attach an existing Stripe install to your current tenant."
---

Use this when Stripe app installation already exists and you need to link it to your current MeterMonitor account.

Typical cases:

- You connected Stripe earlier and signed in with a new MeterMonitor user.
- You are changing Stripe account and want to attach an existing install.

## Required input

- Stripe `account_id` (example: `acct_...`)
- Correct mode: `test` or `live`

## Steps

1. Go to **Settings**.
2. Click **Change Stripe Account**.
3. Select **I already installed Stripe and want to claim existing install**.
4. Choose mode and enter `account_id`.
5. Submit and confirm success.

If claim fails, verify account ID and mode, then retry.
